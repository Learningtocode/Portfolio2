Technology used for this Web site

HTML

HTML or HyperText Markup Language is a standard system for tagging text files to accomplish web sites on the Internet.

CSS

CSS or Cascading Style Sheets is a style sheet language used for describing the look and formatting of a document written in a markup language.

Bootstrap

Bootstrap is a technique of loading a program into a computer by means of a few initial instructions that enable the introduction of the rest of the program from an input device.
